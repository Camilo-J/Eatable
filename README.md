## Getting Started

To get a local copy up and running follow these steps.

### Prerequisites

- Bun

### Installation

1. Clone the repo, or download the zip file

   ```bash
   git clone git@github.com:Camilo-J/Eatable.git
   ```

2. Install NPM packages

   ```bash
   bun install
   ```

3. Run the app

   ```bash
   bun run dev
   ```
